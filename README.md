# Vanderven Systems

Marketing site for Vanderven Systems — websites and AI automation for trades, real estate professionals, and service businesses in the Okanagan.

## Run locally

Open `index.html` in a browser, or from this folder:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Logo animation

The handwritten **V** in the hero is an SVG stroke draw-on (`stroke-dashoffset`) that runs on page load. Source mark: `assets/logo.svg`. Replace the path with a traced version of your final logo for a perfect match.
